# Testing Algorithms (Kinematic Engine)

This directory contains the source code for the algorithms used to calculate structural failure risk and mechanical resonance.

## Core Scripts
* **Fracture_Predictor_v1.py**: The primary engine for calculating Pf (Fracture Probability) from stacking energy and impedance.
* **Extraction_Logic**: Python functions used to parse GRCh38 FASTA files and identify TAC signatures.

## Diagnostics
* **Test_Cases.csv**: A set of controlled sequences used to calibrate the 1.33 resonance threshold.
